some reason it is down and blocked in many countries. People are interested in setting up their own version of this fantastic web service to serve the download enthusiast and make some money by showing ads. But it is very difficult to find a good quality clone script, all available options are paid and hugely expensive. So I decided to make my own clone for the open source community and build this super powerful yet very lite tiny php/javascript script which requires no database support (using my own server side api). This beautifully designed script can serve thousands of traffic daily from a very low powered server (even static web hosting is enough). Ads spot provided for banner ads. Fully SEO friendly and mobile first responsive design will attract you.

Server Requirements: 
1. Apache php with "mod_rewrite".
2. You can use static html web hosting services to host this script, just rename the "index.php" file to "index.html", but sitemap.php will not work.
3. Also you can host the script in surge.sh free static web hosting service. (I an using it, demo: https://torrentz2.1111110.xyz/)
   Just rename the "index.php" file to "200.html" and upload to surge.sh.

Installation:
1. Unpack the .zip package.
2. Open "index.php" with text editor for editing.
3. Find "place top banner ads here" and replace it with your banner ads code in "index.php".
4. Change "siteTitle" variable value in "index.php" with your site name (i.e change 'Torrentz2' with 'myTorrentSite' etc).
5. Upload all the files to your server.
6. {Recomended/Important] Submit your site map to Google Search Console. (sitemap url https://yoursiteurl.com/sitemap.php).
7. For any question, suggestion or help please contact me at "info@1111110.xyz".


8. {optional] Choose any of the languages from the list below and change "siteLanguage" value to change site language. (But Google will always crawl it in english).
   For example 'zh-CN' is for  'Chinese Simplified'.
   Site default language is English (i.e. "en").
